# Development Roadmap

## Milestone 1: Foundation

Monorepo, shared types, Supabase schema, seeded templates, app shells.

## Milestone 2: Authentication

Supabase email, Google, Apple, profile creation, protected mobile routes.

## Milestone 3: Builder

Template creation flow, title, recipient, message, theme, uploads, page CRUD, preview.

## Milestone 4: Publishing

Slug generation, published status, copy link, native sharing.

## Milestone 5: Renderer

Supabase-backed `/e/[slug]`, dynamic page rendering, proposal interaction, watermark control.

## Milestone 6: Analytics

Views, unique visitors, completion rate, average time, button clicks, proposal attempts.

## Milestone 7: Subscription

Free plan limits, Pro unlocks, Razorpay order and payment verification.

## Milestone 8: Production Hardening

Rate limits, abuse protections, upload limits, SEO metadata, error states, QA checklist, deployment docs.
